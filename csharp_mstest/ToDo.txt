整数の閉区間

- [ ] 整数閉区間を示すクラスをつくる
  - [x] [3,8]で作る
  - [x] [-5,0]で作る
- [x] 整数閉区間オブジェクトは下端点と上端点を持つ
	- ただし、上端点より下端点が大きい閉区間を作ることはできない
	- [x] 上端点より下端点が大きい閉区間を作ると、例外が発生する
	  - [x] [8,3]で作ると例外が発生する
	  - [x] [5,4]で作ると例外
	  - [x] [5,5]だと。。。？
- [x] 文字列表現を返せる
	- （例: 下端点 3, 上端点 8 の整数閉区間の文字列表記は "[3,8]"）。
  - [x] [3,8]
  - [x] [-5,0]
- [ ] 指定した整数を含むかどうかを判定できる
- [ ] 別の閉区間と等価かどうかを判定できる
- [ ] 別の閉区間を完全に含むかどうかを判定できる
